Problem Spotlights:

This page features writeups, examples, and so on for problems that I found interesting.
  • Random Numbers - how do computers generate random numbers, and how can we use math to manipulate pseudorandom number generators?
  • Beacon Scanners (Advent of Code 2021) - how can we combine information about the locations of multiple objects from the point of view of different scanners whose own locations we don't know?
  • Run Graphs (November 2021) - what if I ran every day for a month and made some graphs?
  • Normal Rating System - how can we design a rating system like Elo or Glicko, and how can we use it to rank a large list of options?
